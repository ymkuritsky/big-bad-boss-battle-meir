# Battleship Thursday

A new comic-style mystery board game.

First version:
- Choose between Smiley Bone, Phony, and Grandma Ben.
- Pick mystery boxes on a small battleship board.
- Each box starts a small fight with the villain inside, like Big Bad Boss Battle.
- Every box now reveals a villain.
- Special villains include King Dock, MIM the Red Dragon, the Lord of Locusts, big black rat creatures, and Bartlebee the mega rat creature with a club.
- Some new villain names are placeholders, ready to replace or redesign with Meir's next ideas.
